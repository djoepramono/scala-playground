## How to use

Make sure you have `sbt` installed on your computer,
because all you need to do is:

```
sbt run
```

And you will see a set options to choose from. Each options covers a learning
topic. This topic code is available on [src/main/scala](./src/main/scala) while
the accompanying documentation is available at [docs](./docs)
